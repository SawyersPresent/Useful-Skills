# Useful Skills

Curated, shareable AI skills for security workflows.

## Included Skills

| Skill | Focus |
|---|---|
| `byovd-re` | Windows BYOVD driver reverse-engineering and triage workflow |


## Safety Principles

- No malware development guidance.
- No exploit weaponization instructions.
- No persistence/evasion implementation guidance.
- Use only in authorized lab or assessment environments.

## How To Use

1. Copy a skill folder (for example `byovd-re`) into your local skills directory.
2. Invoke it by name in your AI assistant.
3. Follow the skill-specific README and `SKILL.md` instructions.

### Recommendation

It is possible to use a Reverse Engineering MCP server and its recommended for that tool to have its own skills file.
